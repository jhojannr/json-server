## Para empezar

### Librería del Proyecto
- <a href="https://nodejs.org/es/">Node JS (18.16.0)</a><br>
- <a href="https://github.com/typicode/json-server/">JSON Server (0.17.3)</a>


### Instalación Inicial
```
npm install
```

### Compilar e iniciar el Servidor
```
npm run start
```
